# Security Policy

## Supported Versions

This repository currently tracks a single active line on `main`.
Security fixes are applied on `main` first.

## Reporting a Vulnerability

Please do not open public issues for suspected vulnerabilities.

Report privately by email:

- security contact: `security@gaialynk.com`

If that email is unavailable, open a private GitHub security advisory in this repository.

Include:

- Affected endpoint or module
- Reproduction steps
- Impact assessment
- Suggested mitigation (if available)

## Response Expectations

- Initial triage target: within 72 hours
- Valid issues receive a mitigation plan and timeline
- Coordinated disclosure is preferred after patch release
